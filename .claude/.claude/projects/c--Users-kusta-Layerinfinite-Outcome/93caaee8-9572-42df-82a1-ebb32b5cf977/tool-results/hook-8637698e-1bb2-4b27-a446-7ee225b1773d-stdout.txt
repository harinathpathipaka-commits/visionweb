Product Overview\n\nLayerInfinite is a **decision intelligence layer for autonomous AI agents**. It operates as a transparent MCP (Model Context Protocol) gateway proxy positioned between AI agents and their upstream MCP tool servers. Its core function is to observe, score, and optimize which tools an AI agent chooses for a given task, based on historical outcome data.\n\n**The fundamental problem it solves**: AI agents making tool-call decisions have no memory of what worked before. An agent choosing between `github_push_fix` and `git_cli_commit` for a \"fix build\" task has no way to know which historically succeeded more often. LayerInfinite provides that memory.\n\n**Data flow**: `AI Agent --> [MCP Gateway Proxy] --> LI API --> PostgreSQL (Supabase)`\n\nThe gateway intercepts `tools/list` (enriching descriptions with success rates) and `tools/call` (optionally rerouting in auto mode). The critical invariant: `inputSchema` is NEVER modified -- only `description` is enriched. In recommend mode, success rates are woven into the description text. In assist mode, ranked comparisons and directional warnings are added. In auto mode, descriptions are passed through unchanged (the agent should not know LI is mediating).
