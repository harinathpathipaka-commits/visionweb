WHAT IS LAYERINFINITE\n\nLayerInfinite is a **decision intelligence layer for autonomous AI agents**. It operates as a transparent MCP (Model Context Protocol) gateway proxy positioned between AI agents and their upstream MCP tool servers.\n\n### The Core Problem\n\nAI agents making tool-call decisions have no memory of what worked before. An agent choosing between `github_push_fix` and `git_cli_commit` for a \"fix build\" task has no way to know which historically succeeded more often. LI provides that memory.\n\n### How It Works (7 Steps)\n\n1. Agent boots up, requests tools via LI gateway (`tools/list`)\n2. LI forwards to real upstream MCP servers, fetches actual tool lists\n3. LI queries outcome history for current task вҖ” available tools get enrichment\n4. LI returns enriched tool list вҖ” natural language historical scores woven into descriptions\n5. Result returned to agent immediately вҖ” outcome logged async in background\n\n### The Key Insight\n\nLI intercepts `tools/list` which happens BEFORE the agent reasons about which tool to use. By enriching descriptions at discovery time, LI doesn't need to intercept the decision because the decision hasn't happened yet.\n\n---\n\n## 2.
