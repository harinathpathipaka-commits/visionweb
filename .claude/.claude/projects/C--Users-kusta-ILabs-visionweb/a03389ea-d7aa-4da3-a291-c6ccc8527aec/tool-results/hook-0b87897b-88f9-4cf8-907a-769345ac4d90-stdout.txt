WebSocket Wiring Plan\n\n### Current State\n\nThe gateway's `WebSocketServer` has:\n- A `broadcast::Sender<String>` for pushing events to connected WS clients.\n- A `push_event(&self, event: &str)` method.\n- An `event_sender()` method returning the sender.\n\nThe gRPC server's `EventBus` has:\n- `goals: GoalEventBus` — broadcasts `GoalUpdate { goal_id, progress, status, message }`.\n- `eye_reports: EyeReportBus` — broadcasts `EyeReportEvent { eye_name, goal_id, confidence, content }`.\n\n**Problem**: Currently, the gateway's WS broadcast channel and the gRPC server's event bus are completely disconnected.
